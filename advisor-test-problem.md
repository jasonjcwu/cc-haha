# Advisor 测试题目：双向链表实现

## 题目要求

实现一个双向链表（Doubly Linked List）类，包含以下功能：

### 基础要求
1. **节点类（Node）**
   - 属性：`value`（存储值）、`next`（指向下一个节点）、`prev`（指向前一个节点）
   - 构造函数：接受一个值参数

2. **双向链表类（DoublyLinkedList）**
   - 属性：
     - `head`：指向链表头部节点
     - `tail`：指向链表尾部节点
     - `size`：链表长度
   - 方法：
     - `append(value)`：在链表尾部添加节点
     - `prepend(value)`：在链表头部添加节点
     - `insertAt(index, value)`：在指定索引位置插入节点
     - `remove(index)`：删除指定索引位置的节点
     - `get(index)`：获取指定索引位置的节点值
     - `set(index, value)`：设置指定索引位置的节点值
     - `contains(value)`：检查链表是否包含某个值
     - `reverse()`：反转链表
     - `toArray()`：将链表转换为数组
     - `clear()`：清空链表

### 进阶要求
1. 实现迭代器接口，支持 `for...of` 循环
2. 实现边界条件处理（空链表、索引越界等）
3. 添加适当的错误处理和类型检查（TypeScript）
4. 实现链表合并和拆分功能

### 测试用例要求
- 编写单元测试验证所有功能
- 测试边界条件（空链表、单节点、双节点）
- 测试链表反转的性能
- 测试并发操作（如果有）

### 额外挑战
- 实现循环双向链表（Circular Doubly Linked List）
- 添加排序功能
- 实现链表快照功能，可以回滚到某个状态

## 评判标准
1. 代码质量和可读性
2. 边界条件处理
3. 性能优化
4. 测试覆盖率
5. 代码结构和设计模式使用
